# Legacy Root Tools

Thư mục này chứa các script `.py` từng đặt ở thư mục gốc, chủ yếu phục vụ:
- one-off fix,
- kiểm tra thủ công theo từng tình huống,
- utility tạm thời không thuộc luồng runtime chính.

Các script đã chuyển vào đây để thư mục gốc gọn hơn và giảm rủi ro chạy nhầm.

## Danh sách đã chuyển
- `check_chunks.py`
- `create_reference_docx.py`
- `fix_cjk_regex.py`
- `script_fix_prompt.py`
